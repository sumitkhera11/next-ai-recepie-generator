6️⃣ Why Object.entries() is used after reduce

reduce() returns object:

{
  Protein: [...],
  Spices: [...],
  Oil: [...]
}

But React .map() works on arrays.

So we convert object → array:

Object.entries(groupedIngredients)

Result:

[
  ["Protein", [...]],
  ["Spices", [...]],
  ["Oil", [...]]
]

Now React can map:

.map(([category, items]) => ...)
